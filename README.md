# AI Vision Studio 🚀

A modern, fast, and minimalist AI-powered image generator built for the **NASA Stardance Summer Program**. This application allows users to transform text prompts into high-quality digital imagery using cutting-edge AI integration.

## 🌟 Key Features
- **Instant Generation:** Real-time image processing via Pollinations AI API.
- **Modern UI/UX:** A sleek, "Glassmorphism" aesthetic with a professional dark theme.
- **Responsive Design:** Optimized for both desktop and mobile devices.
- **Seamless Export:** One-click high-resolution image downloads.
- **User Experience:** Integrated loading states and animations for a smooth interaction.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Advanced styling with glassmorphism, gradients, and custom animations.
- **JavaScript (ES6+):** Asynchronous API handling and dynamic DOM manipulation.
